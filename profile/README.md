# XSAVO

**Building the tools agents need.**

We're an AI infrastructure lab building the interfaces, APIs, and primitives that enable autonomous agents to interact with the physical and digital world.

Every piece of infrastructure that assumes a human operator will need an agent-native interface. We're building that layer.

---

### Products

| Product | What it does | Status |
|---|---|---|
| **TextAPI** | Global SMS & communication API engineered for LLM consumption. MCP-ready. | `In Dev` |
| **CrewBox** | Multi-agent team orchestration for solo founders. | `In Dev` |
| **Byte** | Headless macOS AI assistant — Telegram, CLI, and desktop in one `.dmg`. | `Experimental` |

### The Lab

We run weekly research sprints on the edges of machine autonomy and coordination:

- **Agent Tooling** — LLM "hand-eye" coordination across browsers, shells, and APIs
- **Prototypes** — Autonomous SDRs, AI-driven DevOps pipelines, stateful memory systems
- **pi-mono** — Our internal monorepo primitives for auth, state, and memory
